Figure 5: trajectory_topology_on_embedding.png
------------------------------------------------------------------------
Trajectory topology inferred by elpigraph overlaid on the shared UMAP embedding.
Numbered circles mark cluster centroids computed as the mean UMAP coordinate of
all cells belonging to that cluster in the SHARED embedding
(adata.obsm['X_umap']), ensuring all methods are visualised in the same

maps numeric IDs to cluster names.

Figure 8: stability_hist_pseudotime_spearman_abs.png
------------------------------------------------------------------------
Distribution of pairwise Spearman |rho| correlations between pseudotime vectors
computed across 20 bootstrap replicates (190 pairs). Each replicate is generated
by sampling 80% of cells with stratification by cell type. The absolute value
accounts for direction ambiguity. X-axis fixed to [0, 1] for cross-method
comparability. Values close to 1.0 indicate high pseudotime stability; values
below 0.7 suggest poor reproducibility.

Figure 9: stability_hist_edge_jaccard.png
------------------------------------------------------------------------
Distribution of pairwise edge Jaccard indices across 20 bootstrap replicates
(190 pairs). The Jaccard index measures the overlap of edge sets (edges
restricted to nodes shared between both replicates). X-axis fixed to [0, 1] for
cross-method comparability. Values close to 1.0 indicate that the same topology
is recovered consistently; values below 0.5 suggest unstable branching
structure. Fixed-edge methods produce discrete bars: J = k/(2n-k).
